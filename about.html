Sobre mí

Desarrollo software con atención a la estructura, la claridad y el valor que entrega a las personas.

Software mantenible para problemas reales

Soy desarrollador de software con un enfoque principal en Java, Spring Boot y la construcción de servicios backend. Trabajo en la organización de lógica de negocio, APIs REST e integración con bases de datos, procurando que cada solución sea comprensible, mantenible y adecuada para el contexto en el que se utiliza.

Mi perfil también abarca desarrollo Full Stack. Esto me permite comprender el recorrido completo de una funcionalidad, desde el modelado de datos y las reglas del sistema hasta la interfaz con la que interactúa el usuario. Utilizo esta perspectiva para tomar decisiones técnicas coherentes y colaborar mejor entre disciplinas.

Me interesa resolver problemas con criterio práctico: entender primero la necesidad, reducir complejidad innecesaria y documentar las decisiones importantes. Valoro el trabajo colaborativo, la comunicación directa y la revisión constante como parte de un proceso de ingeniería responsable.

Mantengo una actitud de aprendizaje continuo para fortalecer fundamentos, incorporar herramientas cuando aportan valor y mejorar la calidad de lo que construyo. Busco participar en equipos y proyectos donde el software sea tratado como un producto que debe evolucionar de forma sostenible.
